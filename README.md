# TSMP3 v2.0

Descarga audio de YouTube en MP3 (320kbps) con una interfaz moderna y portátil.

## 🚀 Descarga
Puedes bajar el ejecutable directo desde la sección de **[Releases](https://github.com/joaquin018/tsmp3/releases)**. No requiere instalación.

## ⚙️ Tecnologías
- **Lenguaje**: Python 3.12+
- **Interfaz**: CustomTkinter
- **Motor de descarga**: yt-dlp
- **Procesamiento de audio**: FFmpeg
- **Empaquetado**: PyInstaller
